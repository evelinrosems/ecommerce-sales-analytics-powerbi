# 📊 Ecommerce Sales Analytics Dashboard (Power BI)

## 🔹 Overview

This project presents an interactive Power BI dashboard analyzing retail sales data.
It helps understand revenue trends, top-performing countries, and overall business performance.

---

## 🔹 Key Insights

* 💰 Total Revenue: **9M+**
* 📦 Total Quantity Sold: **5M+**
* 🧾 Total Orders: **19K+**
* 🌍 Highest revenue generated from **United Kingdom**
* 📈 Peak sales observed in **November**
